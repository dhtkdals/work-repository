
Transfermarkt 선수 정보 크롤러
개요
이 프로젝트는 축구 팬들을 위해 Transfermarkt 웹사이트에서 축구 선수의 정보를 크롤링하는 도구입니다. Transfermarkt은 선수의 시장 가치 및 기본 정보를 제공하는 사이트로, 이 크롤러를 사용하면 선수들의 특징 정보를 수집할 수 있습니다.

사용 라이브러리
이 프로젝트에서는 다음과 같은 파이썬 라이브러리를 사용합니다:

requests: 웹페이지에서 정보를 요청하기 위한 라이브러리.
BeautifulSoup: HTML 문서를 파싱하고 원하는 데이터를 추출하기 위한 라이브러리.
pandas: 데이터를 효율적으로 다루기 위한 라이브러리.
time: 일정한 시간 동안 프로그램을 일시 정지하기 위한 라이브러리.
데이터 저장
크롤링한 선수 정보는 다음과 같은 리스트로 저장됩니다:

number: 선수 번호.
name: 선수 이름.
position: 선수 포지션.
age: 선수 나이.
nation: 선수 국적.
team: 소속 팀.
value: 선수의 현재 시장 가치.
작동 방식
프로그램은 반복문을 통해 네 개의 페이지를 가져옵니다. 각 페이지의 URL을 생성하고 해당 URL에 요청을 보내서 페이지 내용을 가져옵니다.

BeautifulSoup을 사용하여 페이지 내용을 파싱합니다. 선수 정보가 담긴 HTML 테이블 행을 찾아서 필요한 정보를 리스트에 추가합니다.

각 페이지를 스크래핑한 후, 1초 동안 대기합니다.

모든 페이지를 스크래핑한 후, 선수 정보를 Pandas DataFrame으로 저장합니다.

사용 방법
사용자가 원하는 선수의 정보를 검색하기 위해 find_player_info 함수를 사용할 수 있습니다. 이 함수는 입력받은 선수명으로 DataFrame에서 해당 선수 정보를 찾아 반환합니다.

무한 루프를 통해 사용자에게 선수명을 입력받습니다. 입력한 선수명으로 정보를 검색하고, '종료'를 입력하면 루프가 종료됩니다.
